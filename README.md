# 🚲 Bike Sales Report System (Java)

## 📌 Overview
This project is a simple Java console application designed to simulate a bike sales reporting system. It allows users to input multiple customer purchase records and displays structured details such as bike name, model, manufacturing year, and price.

The application demonstrates core Object-Oriented Programming (OOP) concepts along with basic input handling in Java.

---

## ⚙️ Features
- Accepts multiple customer purchase records
- Stores and manages bike details using a class
- Displays formatted output for each entry
- Generates a simple customer ID for each record
- Easy-to-understand and beginner-friendly code

---

## 🧠 Concepts Used
- Classes and Objects  
- Methods  
- String Manipulation (`split`)  
- Arrays  
- Loops (`for loop`)  
- Input Handling using `Scanner`  

---

## 🖥️ How It Works
1. The program asks for the number of entries.
2. Each entry is given in the format:
   ```
   Name,Model,Year,Price
   ```
3. The system processes each input and displays:
   - Customer ID
   - Bike Name
   - Model
   - Year
   - Price

---

## 💻 Sample Input
```
2
Yamaha,R15,2022,180000
Honda,CBR,2021,150000
```
